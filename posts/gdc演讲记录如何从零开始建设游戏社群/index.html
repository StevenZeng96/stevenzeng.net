GDC演讲记录：如何从零开始建设游戏社群


2019年09月12日

本文内容摘记自 GDC 2019年演讲《Building a Community for Your Game from Scratch》 如需查看原文请访问 Youtube 查看演讲实录


对于游戏业界而言，一直存在一种帮助电子游戏销售的“灵方妙药”。


在过去，大家可能会说一定要让媒体知道你的游戏并开始讨论你的游戏，之后，大家开始对视频领域猛追猛打“一定要让 PewDiePie 玩你的游戏”成为了大家挂在嘴边的事情，随后风向又改到了直播领域…


必须要承认的是，以上的这些内容确实卓尔有效，但根据我以及我周围人的认知来看，当前游戏业界的“灵方妙药”便是“社群建设”。“社群”虽然一直存在，但直至今天，“社群建设”才成游戏业界中首要大事，我今天的主要内容便是和大家分享一下关于“社群建设”方面的事情。

主讲人介绍

Mike Rose

  • 前游戏行业作者（曾为 Gamasutra, Kotaku 等游戏媒体供稿）
  • 先就职于 No More Robots （游戏发行商，旗下发行的游戏《速降王者》 曾红极一时）


需要坦言的一点是，虽然当前对于“如何管理社群”存在着大量的参考意见，但“如何从零开始建设游戏社群”却存在大量的空白。所以今天我要跟大家具体分享的内容便是“我是如何一步步从零开始建设游戏社群的”。

你需要哪些工具

我想只要你在游戏行业工作，那么以上几款工具想必你已经多多少少有所接触，没有什么“不为所知的神秘工具”。


  • Discord: 社群运营工具，我们通过 Discord 管理我们的游戏社群
  • Discordbots.org: 机器人创建工具，创建机器人帮你做各种各样事情
  • Google Forms：表单工具，收集玩家信息并开跑项目
  • Mailchimp：邮件工具，给参与者邮寄信息
  • Steamworks：开发工具，创建 Beta 测试

为何选择 Discord


那么首先的问题就是为何要选择“Discord”。


至于为何要选择 Discord，原因便是“只要一个用户使用了 Discord，那么他有很高的概率是一个喜欢电子游戏的人”，在这个游戏平台上，用户购买你所制作游戏的概率会高很多。


而至于另一个方面的原因便是，如果有人加入到了你的 Discord Server（译者注：Discord Server 的加入是通过链接访问的，因此可以公开到任何地方），那么点击链接加入的他一定是对游戏感兴趣的人。


除此之外，Discord 还有不少方便的功能可供使用，你可以使用代码进行管理，而如今已经颇具规模的 Discord 社群甚至能够为不懂代码的你提供方便有效的机器人进行社群的组织管理。


总而言之，Discord 如今正是一个建设游戏社群的不二之选。

步骤一：建设你的 Discord Server


1.取好名字并创建好你的 Discord Server

2.将 Discord Server 完成认证

3.按标准模版建设 Discord Server

完成认证的好处：

  • Server 将会出现在 Discord 游戏服务器列表中；
  • 允许你进行链接与图片方面的自定义；
  • 当前完成认证的游戏数量有限，完成认证将会给你提供一定量级的游戏曝光；

1. 你需要建设哪些频道

  • 欢迎频道：首先和其他人一样，我也选择建设了“Welcome”频道，这是一个新人进入 Server 之后首先会进入地方
  • 公告频道：其次便是“Announcements”频道，这是一个只有你才能够说话的地方，在这里你可以通过 pin 的方式通知到你 Discord Server 中的所有人。
  • 游戏讨论频道：还有就是《降速王者》部分的频道，这是玩家们用于游戏讨论的地方

我还记得首次为《降速王者》建设 Discord Server 的时候，刚刚公布就有大量玩家涌入。


当时我对于 Discord 还一无所知，而玩家们也会通过各种你不知道的神奇套路滥用你的 Discord Server，完完全全的混乱不堪，而这样的情景也将会在你的身上出现。不过这样现象的好处便是你能够从中有所收获，了解到 Discord 运营的方方面面。当然了，Discord 也有非常多的文档供你查阅，帮助你防止诸如此类的事情发生。

2. 特殊的迎新频道

除了上述的基础频道之外，我还会在 Discord Server 中创建一个类似“迎新频道”的地方，这种地方的作用就是强迫新用户尝试用更多的方式参与社群互动。简而言之就是“强迫用户在正式加入 Discord Server 之前进行一些操作”。



拿《降速王者》的 Discord Server 为例，所有进入 Discord Server 的人只会看到一个“选边站”的频道并会在频道中阅读一段“你需要输入指令以加入服务器”的文字（《降速王者》一共三个队伍，这也是为何做了“选边站”频道的原因）。


当玩家输入正确的指令后，“选边站”频道就将会在玩家的视野中消失，而真正进行讨论的频道和玩家所选择的团队专属频道将会提供给玩家自己（顺带还会通过不同的颜色标记不同团队的玩家）。


我们做这样的目的便是**希望能够让进入我们 Discord Server 的玩家感受到自己真正成为了 Discord Server 中的一员。**要知道，如果你没有这样的操作，对于玩家们而言他们可能只是进来匆匆扫一眼过后又匆匆离开。但这样让玩家进行第一步操作的行为将会迫使玩家进行更多的操作。

小知识：Dynobot 完成自动化工作


Dyno 是我完成上述操作的工具，Dyno 的代码非常简单，你可以通过图片中看到我写的一些代码，我便是这样完成了“选边站”频道中的功能。

小知识：使用 Discord 权限管理



还有就是 Discord 权限管理，通过这样的设置能够保证只有完成选边站的玩家才能看到服务器中的正式讨论内容。

步骤二：如何保持用户活跃



在以上的步骤完成之后，你需要考虑的事情便是如何保持社群中的用户持续活跃在群组之中。对于《降速王者》而言，在用户完成“选边站”，用户名字完成颜色标记之后，我们便开始了每周的团队对抗比赛，比赛并不会是大家所想的非常高级的玩意，比如“俳句比赛”之类傻活动，只需要保持玩家能够有活动参与即可。


在我们为《降速王者》宣布了这样的活动过后，我们的活动非常的成功，而这样活动也不需要我们进行太多的操作，周一上来宣布一下活动，周五上去公布一下活动结果就行。当然了，对于每周获胜的团队我们也会有相应的奖励，比如游戏内的奖励或是 Discord 的贴纸等等，玩家们对此非常喜欢。


在《Not Tonight》中他们选择了一套“赚钱“游戏，玩家可以通过输入进行“上班”/“下班”赚取资金，玩家们可以通过这样的活动兑换游戏内道具或是游戏本体等活动奖励，在这样的机制下即使一周不闻不问，玩家照样自得其乐。 在《Hypnospace Outlaw》中，他们在 Discord 中做了一个类似 90 年代互联网的游戏，虽然非常有趣，但耗时耗力，并没有太多的建设必要。 译者注：重点在于如何因地制宜，根据游戏的特色构造出一个有趣的 Discord Server 而非生搬硬套

步骤三：创建 Beta 测试



至于 测试部分则非常简单，你只需创建一个 Beta 测试的频道，然后将 Google 表单链接丢过去，要求玩家填写名称和邮箱地址即可。在这样的情况下，你就有了一个可以让玩家们讨论的 Discord Server，同时也有了一个可以告诉玩家们最近更新情况的邮箱地址。

步骤四：“开闸放水”

以上的三个步骤均是在幕后完成，在完成了以上的准备工作之后你便可以准备开始“开闸放水”了。准备将 Discord 链接传递给尽可能多的玩家，告诉玩家们只要加入 Discord 你便有机会加入到“游戏测试”中，通过“游戏测试”吸引玩家的加入。


我见过不少人创建了自己的 Discord Server，然后跟大家说“我 Discord 搭好了，快来一起聊聊吧！”，可惜的是这样的方式并不有效。


“游戏测试”的方式非常有效，通过这样的方式你可以吸引到成千上万的人加入你的 Discord Sever。


我在进行 Discord 运营中发现有免费赠送 Steam key 的网站，虽然网站上的人只是想要免费拿到游戏 Key 而已罢了，但无论如何这样的用户在拿到我们 Beta 测试的机会之后，提供的游戏反馈是有效的，同时 Discord 也给他们一个直接和开发者对话的窗口。

小知识：Checklist



需要做的事情：

  • 通过 Steam 创建单独的 Beta 测试用游戏
  • 选择一个周末开启宣传活动
  • 通过 Mailchimp 通知你的用户
  • 通过 Discord 收集游戏 Bug
  • 通过 Google 表单 收集用户反馈

同时不要忘记请求参与 Beta 测试的玩家将游戏加入愿望单！达到一定数量（目标八千愿望单）的愿望单后将能将你的游戏在发售之时于 Steam 首页上进行展示！


Checklist:

  • 在 Discord Server 上拥有数千用户
  • 当前的全部用户已经游玩过游戏的早期版本并期望游玩到更多内容
  • 用户对你和你的游戏颇有兴趣并期待更多内容的到来

在完成了以上的所有的内容之后，你只需要在保持玩家活跃的情况下等待游戏正式发售即可。

步骤五：通过小游戏活跃用户

在发售前期，我们会通过与游戏本体紧密相关的小游戏活跃现有用户。活动目标是在游戏发售前活跃尽可能多的社区内用户并让用户感知到自己是社区中的一份子。


在《Not Tonight》中，我们做了“首相选举”以及“间谍”等类似桌游的小游戏，玩家在这些活动中会一同协作完成目标。

当然，也是通过这样的小游戏的方式，玩家将会好奇这些小游戏的元素将会如何运用在正式版本的游戏当中。

小知识：Checklist



  • 如今你已经拥有了一帮子敲锣打鼓等待游戏上线的玩家了
  • 这些玩家们将会在游戏上线的时候通过疯狂的好评刷爆你的 Steam 评测
  • 游戏发售时带来的购买量将会帮助游戏冲击 Steam 销售排行榜并为你带来更多的购买量
  • 而游戏萌新此刻加入的社群已经是一个乐于助人的组织了
