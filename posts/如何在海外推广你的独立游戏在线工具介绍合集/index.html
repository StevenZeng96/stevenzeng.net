如何在海外推广你的独立游戏：在线工具介绍合集

本文原刊于 Medium.com 由 Thomas Reisenegger 撰写 中文版本由 2P.com Steven 译制，翻译转载已授权

推广游戏本就是一件不容易的事，如果你不巧是一个独立游戏开发者？那么这样的事情就变得难上加难了。

对于游戏推广而言，其实说实话并没有什么独门配方，也没有什么窍门能够帮你一蹴而就，不过好在市面上有不少趁手的工具能够帮助你一把，让你的游戏推广变得更为简单。

所以我觉得，不如就把这些所有我喜欢的网络工具放到一个列表当中，帮助各位推广自己的游戏。在此可以声明，本列表中的所有工具我都有使用过，近两年里，我更是在我的公司 Future Friends Games 里使用他们推广了诸多独立游戏。

另外一提，如果你觉得本列表有所疏漏，请通过 Twitter 与我（@olima）联系，我将定期更新本列表中的内容。

一、Steam 工具

1. ICO Steam Newsletter/每周 Steam 游戏周报

ICO Steam Newsletter

众所周知，在 Steam 平台上，每天都有成千上万的游戏进行发售。

即便有如此数量的新游戏，但知道 Steam 热销游戏的人可以说是寥寥无几（3A 大作除外）。这样的情况导致了远近驰名的独立游戏与媒体宠儿之间产生了巨大的脱节。

毕竟，对于 Steam 平台而言的，成功的不二法门在于算法，而非 PR 稿件的分发情况。

如果你想要了解 Steam 上的热销游戏？除了大家都懂的 Steam 畅销排行榜外，ICO Steam Newsletter 是你的最佳选择，这款免费的 Steam 游戏周报能够帮助你了解 Steam 上当前正在热销的游戏。

这份游戏周报会以评论者的多寡罗列 Steam 上新发售的游戏，帮助你了解本周新游戏的销售情况。

我们团队一般会用评论人数x80的方式（一般情况下可用）衡量一款游戏的销售成绩。如今，我们已经通过这份游戏周报养成了每周查看顶尖游戏的习惯并从他们的商店设置与发售策略中进行学习。

2. Steam.db/ 获悉玩家数量、价格等的网站

Steam.db

现在有不少从 Steam 上抓取数据的网站，但在这些网站中，免费使用且无需登录的SteamDB 是我的最爱。

在 Steamdb 上，除了将 Steamcharts.com 的数据（展示每款游戏当前活跃的玩家人数）与 Steamspy（预估游戏拥有者数量）的数据相结合外，Steamdb 丰富的数据能够让你洞悉一款游戏的方方面面。

不过需要你注意的是，解读 Steamdb 上的数据需要一定的练习与基础，但在 Steamdb 上花费时间练习是有价值的。当你学成出山时，能够清晰的了解其他游戏的动向。

3. Augmented Steam / 在 Steam 商店页面上直接显示玩家数量等情况的插件

Augmented Steam

如果你觉得使用 Steamdb 有些麻烦？那么安装 “Augmented Steam” Chrome 插件会是个不错的方法。

一旦安装完成后，你将能够在每个游戏的 Steam 商店页面上直接看到来自 Steamcharts 与 Steamspy 的数据，除此之外，来自 Opencritic 的综合评分以及历史销售情况也将帮助你的查阅。不过需要注意的是，本插件只有在使用 Chrome 浏览器时才会生效，并不会在 Steam 客户端上进行展现。

强烈推荐使用！

4. Steam Tags Query / 为游戏查找最佳的 Steam 标签工具

Steam Tags Query

发掘正确的 Steam 标签能够帮助你的游戏在“”一栏中出现，如果操作得当（再来一点运气的加持）将你带来大量的购买与愿望单的新增。

如果你不太确定哪些标签适合你的游戏？免费的 Steam Tags Query 将会帮你一把。在这个工具中，你只需要将你的游戏 Steam 页面丢入其中，它就将帮你适合的标签。

虽然我推荐你自己挖掘 Steam 平台上大量的游戏标签，但通过工具，你能够有一个很好的开始。（且有时候能够给你些意想不到的惊喜）

二、媒体工具

Similarweb / 看看网站流量有多大

和知名的 Alexa 类似，Similarweb 能够评估一个网站的流量大小。由于 Similarweb 能够给你反馈一个具体的流量结果，我对于这款工具热爱无比。如果你想要了解一个网站的流量大小，你可以直接访问他们网站或是安装 Chrome 插件。

对于我自己而言，我习惯在进行合作前通过这样的流量查询工具判断对方的规模，操作起来非常方便。

三、社群管理工具

1. Loomly / 定时发送内容至社交媒体平台

Loomly

如果有多人运营你的社交媒体平台账号（如 Twitter、Facebook、Instagram 等），反复的账号切换将会让你的草稿箱混乱不堪。

在 Loomly 上，你可以通过这个集中站点统一管理你的社交媒体账号，撰写、编辑、评论、定时发布等诸多功能将帮你打理多个平台的账号。

Loomly 非常方便，但请注意大多数功能需要进行付费使用。

2. Remove.bg / 轻松抠图工具

Remove Background from Image – remove.bg

如果你想要将图片上的背景抠掉只留下人物或物品本身且不想使用专业工具，那么 Remove.bg 将会是你的不二之选。

Romove.bg 工具可以免费使用且无需登录，抠图精确且行动迅速。
AI 万岁！

3. Tweetdeck / 定时发布推文并追踪你的游戏

Tweetdeck

对于一个活在 Twitter 上的我而言，最爱的工具莫过于 Tweetdeck 了。

Tweetdeck 是一个 Twitter 出品的另一个网页版客户端，但相较于原版而言，Tweetdeck 非常多拥有适合 PR 的功能。

对于初学者而言，使用 Tweetdeck 你可以：

  1. 定时发布推文（超方便）
  2. 多个账号间进行轻松切换/回复
  3. 你可以专门设定一列信息流专门盯着某个搜索结果。比如我会设定如“塞尔达荒野之息”等搜索结果，这样你就能关注到文章与 Youtube 视频的动态信息而非依赖手动 Google 了！

4. Vid IQ / 查看 Youtube 视频的更多信息

Vid IQ

Youtube 搜索向来十分神秘，但它能够帮助你游戏获得更高的关注量并触及更大的人群量。在我看来，给你的视频打上标签并给视频冠以 SEO 友好型的标题将会是从平台手上获取更高流量的免费途径。

那么如何做到这一点？如何了解驱动 Youtube 背后的算法呢？其中的一个方案便是关注成功视频背后的标题与标签。VidIQ 提供的免费 Chrome 插件便能够为你做到这些。

5. Hashtagify / 查看 Twitter 上当前热门的话题标签

Hashtagify

虽然在是否要使用标签方面有着大量的争执，但如果你需要使用标签的话，Hashtagify 这样的网站将会是你的助手。

在这个网站上，你能够查询每个话题标签下信息的活跃度，帮助你在诸如纠结使用 #indiedev 还是 #gamedev 的情况下作出判断。

四、Twitch 工具

Twitchmetrics / 轻松找到 Twitch 主播的方法

Twitchmetrics - Streamers, Teams, & Games

Twitch 非常优秀，但是在这个直播平台上找到适合的主播却是件不容易的事情。

对于 Twitchmetrics 而言，虽然它并不能够解决你当前拥有的问题，但是通过查询主播玩过的游戏，它能够帮你以更加容易的方式找到适合自己的主播。

五、GIF 制作工具

1. Ezgif / 最棒的 GIF 处理工具

Ezgif

大爱 GIF 图！我甚至给我们所有合作过的游戏一一制作了 GIF 作为宣传使用！（这方面，我甚至在 GDC 上做过相关的演讲！）

Ezgif 是一个被完全低估的 GIF 制作网站，以及提供免费使用这一点简直匪夷所思，但咱们抛开这个问题不管的话，这个工具是十分好用的！

不需要登陆的设计和大量可用的功能让它别具魅力，压缩图片大小，拉升缩放图片，图片裁剪，视频转换一切均有！强烈推荐！

2. ScreenToGif / 录屏制作 GIF

ScreenToGif

制作 GIF 最简单的方法莫过于直接录制你的屏幕，而录制屏幕的工具中最简单的方式莫过于 ScreenToGif。

这个免费工具允许你使用小窗口的方式录制 Gif，无论这个窗口底下发生了什么都会忠实地记录。

3. Giphy / 通过 Youtube 视频制作 GIF 的绝妙工具

GIPHY | Search All the GIFs & Make Your Own Animated GIF

如果你已经将你的游戏上传至 Youtube了，然后你又想通过 Youtube 视频快速出个 GIF？那么 Giphy.com 将是你的最棒选择。

操作方式十分简单，你只需要将视频的链接进行粘贴即可，设置好时间戳，然后你的 GIF 就制作完成了！

六、其他工具

1. Dereck-lieu.com/ 游戏视频预告专家

Video Game Trailer Specifications — Derek Lieu - Game Trailer Editor

如果你将自己的游戏上传到大型游戏商城（如 Nintendo eShop、Playstation Store 等）中，那么你一定知道几种视频之间不同的规矩（视频大小，开头处的 logo 展示等等），这意味着当你每当你需要将游戏上传或接受平台推荐时，你需要按着各平台的规矩来，而且有时候各个地区的要求均各有不同。

不过好在 ，预告片负责专家 Derek Lieu 将所有的要求放到了他的网站上，大家可以各取所需。

2. Mailtester/看看你的邮件是否统统丢到垃圾邮件去了

Newsletters spam test by mail-tester.com

建设电子邮件投递名单是独立游戏营销中不可缺少的部分。如果你正在考虑选择邮件邮寄服务的话，Mailtester 是我的推荐首选。

在 Mailtester 中，你可以通过简单数步的设置，完成数步简单的设置后，你通过发送测试邮件的方式了解你的邮件是否会被丢到垃圾邮件中去。除此之外，该网站上还十分贴心地给出邮件发送的指导意见，帮助你快速提升邮件投递成功率。

3. Kicktraq & Biggercake / 看看 kickstarter 上更多的数据

如果你恰好在弄 Kickstarter 众筹的话，那么这款 Kicktraq 这款免费 Chrome 工具我一定要向你推荐！（它还有网站！）除此之外，Biggercake.com 同样也有不少众筹方面的数据值得大家进行参考。

在这些工具的帮助下，你不仅能够看到众筹项目的每日进展情况，同时还能帮助您追踪项目进展并帮助你预估众筹成功时间。

3. GDC Vault / 成千上万的游戏演讲

GDC Vault

GDC Vault 和它 Youtube 频道是你寻求诸如营销、公关乃至社群管理方面知识的宝库！虽然不少内容均带有付费属性，但是仍有不少视频内容现在以免费性质对外开放观看。

4. Cooltext.com/来个火焰效果的文字吧

Cool Text Graphics Generator

如果你想知道火焰文字是如何制作的？那么 cooltext.com 是你的不错的选择。
