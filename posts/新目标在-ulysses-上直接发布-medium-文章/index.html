新目标，在 Ulysses 上直接发布 Medium 文章

仔细一想已经有很长一段时间没在 Medium.com 上发文章了。

因为此前将 XMLinkUP 的文章从 Medium 迁移到了 Wordpress，然后建立了 xmlinkup.com 专门用来存放 XMLinkUP 的文章，Medium 这边就显得干净了不少。既然空着，那就不如放点自己随手写的玩意吧？说干就干。

本地写作！

不过对于我这种折腾来折腾去的人而言，首先第一个要解决的事情就是在 Medium 上面直接使用日常使用的 Ulysses 完成撰写任务。

至于为何“要用 Ulysses 编辑”，这一点相关的讨论那就相当于选择要在“本地”完成你的写作任务还是选择在“云端”完成你的写作任务，两者吵起来估计能够吵上三天三夜。

对于我个人而言，这两种写作方式各有各的好处，主要看你自己的使用习惯，而我更加喜欢在本地完成所有的撰写任务。

如何操作

至于如何操作这就是一件非常简单的事情了！

由于 Ulysses 自身支持 Medium 的文章发布功能，因此你需要做的事情只有前往 偏好设置 - 账户 - 添加 Medium 账户 然后填入 Intergration Token 即可。

但是由于 Medium 站并非对所有人都提供 Token，因此在这里需要你前往 Medium 的帮助页面与客服联系完成接下来的操作。

具体操作便是提交 API 请求工单，然后将你的 Email 地址、Medium Username 以及申请理由逐项填入即可。

我的申请理由如下，供您参考 Hey there, I want to use Medium form Ulysses. Can I get the Medium API access? I want to use the Medium as my blog again.

随后来自 Medium 的团队将会为你开启 API 权限，这样你就可以直接在 Setting Page 看到你的 Integration Token 了。之后填入 Token 即可完成全部操作了！

没错，这篇稿子其实就是在拿到 Token 之后写的，就是想看看能不能发了🤪
